    Наименование: LudereZone


    Краткое описание:
LudereZone — это современное веб-приложение для продажи игрушек. Платформа позволяет пользователям просматривать наиболее значимые характеристики для выбора игрушек, фильтровать каталог, добавлять товары в избранное и управлять профилем. Административная панель обеспечивает полное управление ассортиментом.

    Целевая аудитория:
Родители с детьми, молодые пары, подростки.

    Ключевой функционал:
Каталог с фильтрацией: Просмотр игрушек с фильтрацией по типу, категории, цене и производителю.
Детальная карточка товара: Отображение характеристик, галереи изображений и отзывов.
Личный кабинет: Управление профилем, история просмотров.
Избранное: Сохранение понравившихся товаров в отдельный список.
Поиск: Полнотекстовый поиск по названиям и описаниям.
Админ-панель: операции над товарами и управление пользователями.
Аутентификация: Регистрация, вход, восстановление пароля.

    Структура приложения:
Главная	(/)	Страница с баннерами, списком популярных игрушек и новинок.
Аутентификация	(/auth/*)	Страницы входа, регистрации, восстановления пароля.
Каталог	(/catalog) Сетка товаров с фильтрами (по категориям, цене, популярности) и сортировкой.
Детальная страница	(/catalog/:id) Полная информация об игрушке: фото, характеристики, описание, кнопка "В избранное" и "В корзину".
Ошибка	(/err)	Страница с выводом ошибки.
Избранное	(/fav)	Список товаров, которые пользователь отправил "В избранное".
Личный кабинет	(/prof)	Данные пользователя, избранное, смена пароля.
Сравнение	(/compare)	Сравнение технических характеристик игрушек (выбранных пользователем).
Админ-панель	(/admin)	Управление товарами (добавление/редактирование), просмотр списка пользователей.

    Технологический стек:
Frontend: Фреймворк: React + TypeScript (TypeScript обеспечивает строгую типизацию на фронтенде).
Роутер: React Router v6 (Стандарт для навигации в React-приложениях).
State-менеджмент: Redux Toolkit + RTK Query (RTK Query автоматически управляет кэшированием, загрузкой и состоянием ошибок для API запросов).

Backend: Язык: Python 3.11+ (Современная версия с улучшенной производительностью, поддержкой асинхронности и новыми возможностями типизации).
Фреймворк: FastAPI (Асинхронный, встроенная валидация через Pydantic, высокая производительность)
База данных: MySQL

    __tablename__ = "users"
    
    id = Column(Integer, primary_key=True, index=True)
    email = Column(String(255), unique=True, nullable=False, index=True)
    password = Column(String(255), nullable=False)
    name = Column(String(100))
    role = Column(Enum(Role), default=Role.USER)
    created_at = Column(DateTime(timezone=True), server_default=func.now())
    updated_at = Column(DateTime(timezone=True), onupdate=func.now())

    created_at = Column(DateTime(timezone=True), server_default=func.now())
